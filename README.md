# 유용한 git 명령어
```
- staging 명령
git add .

- stage 상태 확인
git status

- local repository로 이동
git commit -m "커밋 메시지"

- local repository(commit 메시지 확인)
git log

- git commit 후, 취소하기(unstage 까지)
git reset --soft HEAD~1

- git add 후, 취소(unstage)
git restore --staged .  
```
